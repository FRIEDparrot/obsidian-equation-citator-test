This repository is test files for [Equation Citator](https://github.com/FRIEDparrot/obsidian-equation-citator) plugin.

If you want to add more test files or modify some existing test files, please create a issue at [main plugin repository](https://github.com/FRIEDparrot/obsidian-equation-citator/issues) first, describe what you want to test. Then you may also create a PR with test files in this repo.

## Test file format
always use `.ts` for test file extension.
